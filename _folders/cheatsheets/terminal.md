# GNOME TERMINAL CHEATSHEET

## GNOME Terminal Keybindings
- **Ctrl + Shift + T**: Open a new terminal tab.
- **Ctrl + Shift + N**: Open a new terminal window.
- **Ctrl + Shift + W**: Close the current terminal tab.
- **Ctrl + Shift + Q**: Close the current terminal window.
- **Ctrl + Shift + C**: Copy selected text.
- **Ctrl + Shift + V**: Paste copied text.
- **Ctrl + Shift + P**: Open the preferences window.

## Bash Keybindings
- **Ctrl + A**: Move the cursor to the beginning of the line.
- **Ctrl + E**: Move the cursor to the end of the line.
- **Ctrl + U**: Delete from the cursor to the beginning of the line.
- **Ctrl + K**: Delete from the cursor to the end of the line.
- **Ctrl + W**: Delete the word before the cursor.
- **Alt + Backspace**: Delete the word before the cursor.
- **Ctrl + Y**: Paste (yank) the last deleted text.
- **Ctrl + L**: Clear the terminal screen.
- **Ctrl + R**: Search the command history.

## Navigation and Editing
- **Alt + F**: Move the cursor forward by one word.
- **Alt + B**: Move the cursor backward by one word.
- **Ctrl + F**: Move the cursor forward by one character.
- **Ctrl + B**: Move the cursor backward by one character.
- **Ctrl + D**: Delete the character under the cursor.
- **Ctrl + H**: Delete the character before the cursor.
